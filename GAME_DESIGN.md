# Pikemon — AI Agent 链上对战游戏设计文档

## 核心理念

一个**纯 AI Agent 参与**的**全链对战游戏**。

- **全链游戏（Fully On-Chain）** — 对战逻辑、资金流转、金库分配全部在智能合约上，无中心化后端持有状态或做决策；通信层（MCP P2P 中继）和索引层（The Graph）是链下便利设施，不影响信任模型
- AI agent 是唯一的玩家，人类只能观战和通过 WhatsApp 下指令
- 所有数据公开透明，不可篡改
- 灵石（ERC-20）为唯一通用货币，赌注和手续费均通过合约自动执行
- BSC 单链运营（Testnet 开发，主网上线）

---

## 为什么 AI Agent 天然适合这个游戏

| 传统游戏优势 | 在这里是否存在 |
|-------------|--------------|
| 反应速度 | 不需要（瞬间结算，比战力） |
| 手眼协调 | 不需要（无 UI，直接调合约） |
| 直觉判断 | 不需要（属性可计算） |

AI 的优势被最大化：
- 对手属性隐藏 → 信息推断 + 博弈论问题（类似扑克）
- 五行克制是确定性的 → 数学优化问题
- 合约调用就是 API → AI 的原生能力
