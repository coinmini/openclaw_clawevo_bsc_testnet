# 洞天系统

洞天是修仙者在天地间开辟的**私人修炼场所**，内有浓郁灵气，天然适合闭关修炼。
随着修为精进，洞天可升华为福地、灵地，修炼效率不断提升。

> 洞天 ≠ 宗门（宗门是社交+团队增益，洞天是个人修炼空间）
> 洞天 ≠ 装备（洞天是永久性基础设施，不可交易）
> 洞天 = 修仙者专属「私人赛道」，越早开辟越早受益
> 注意：洞天需要每日维护费维持灵气，欠费会导致降级（详见「维护与降级」）

## 洞天三阶体系

| 阶层 | 解锁条件 | 闭关修为速度加成 | 道心/气运被动加成 | 开辟/升级消耗灵石 |
|------|---------|--------------|-----------------|----------------|
| 无（初始）| 练气期～筑基期 | 基准 ×1.0 | — | — |
| 🌿 **洞天**（一阶）| 金丹期突破 | ×1.2（+20%）| — | 500 灵石（开辟）|
| 🏔️ **福地**（二阶）| 元婴期突破 | ×1.4（+40%）| 道心/气运各 +1% | 2,000 灵石（升级）|
| ✨ **灵地**（三阶）| 化神期突破 + 洞天累计闭关 4000h | ×1.6（+60%）| 道心/气运各 +2% | 8,000 灵石（升级）|

```
洞天速率说明：
  × 1.2 = 同等时间内，修为获取量增加 20%
  × 1.4 = 同等时间内，修为获取量增加 40%
  × 1.6 = 同等时间内，修为获取量增加 60%

→ 闭关无时间上限，一次 start 持续闭关直到主动 end 或被 PK 打断
→ 洞天加成不影响灵石消耗（消耗量不变，效率提升）
→ 洞天加成不影响战力属性（只加速修炼进程）
```

## 解锁与升级条件

### 开辟洞天（一阶）

```
条件：
  - 境界达到金丹期（任意小境界）
  - 拥有 500 灵石

流程：
  1. 调用 CaveHeaven.open()
  2. 合约验证境界 ≥ 金丹期
  3. 扣除 500 灵石 → 游戏金库
  4. 链上记录：tier = 洞天，openedAt = 当前时间戳

效果：
  → 后续每次闭关自动享受 ×1.2 修为速率
  → 无需额外操作，洞天加成持续生效
```

### 升级福地（二阶）

```
条件：
  - 境界达到元婴期（任意小境界）
  - 当前 tier = 洞天（一阶）
  - 拥有 2,000 灵石

流程：
  1. 调用 CaveHeaven.upgrade()
  2. 合约验证境界 ≥ 元婴期
  3. 扣除 2,000 灵石 → 游戏金库
  4. tier = 福地，upgradedAt = 当前时间戳

新增效果：
  → 闭关速率提升至 ×1.4
  → 道心/气运积累速率 +1%（每次闭关积累量额外 ×1.01，实时生效，合约读取）
```

### 升级灵地（三阶）

```
条件：
  - 境界达到化神期（任意小境界）
  - 当前 tier = 福地（二阶）
  - 洞天内累计有效闭关时长 ≥ 4000 小时（链上记录）
  - 有效闭关定义：单次闭关时长 ≥ 4 小时才计入累计（block.timestamp - startTime >= 4 hours）
  - 每次结算时：cultivationHours += 实际时长（无上限）
  - 拥有 8,000 灵石

4000 小时累计闭关条件的意图：
  → 激励真实使用洞天，而非仅为数字升级
  → 正常玩家 14~16h/天闭关：4000h ÷ 14h ≈ 286 天，÷ 16h = 250 天
  → 从金丹期开辟洞天起即开始累计（含金丹→元婴→化神全程）
  → 4h 最低时长防止脚本短循环刷时长（短于 4h 的闭关不计入累计）
  → 累计小时制天然防刷：无论拆成多少次，总时长不会凭空增加

流程：
  1. 调用 CaveHeaven.upgrade()
  2. 合约验证境界 + cultivationHours ≥ 4000
  3. 扣除 8,000 灵石 → 游戏金库
  4. tier = 灵地

新增效果：
  → 闭关速率提升至 ×1.6
  → 道心/气运积累速率 +2%（替换福地 +1%，每次闭关积累量额外 ×1.02）
```

## 修炼收益公式（三重叠加）

```
洞天加成与宗门、道侣三重独立叠乘：

修为获取速率 = 基础修为速率
              × 洞天灵气倍率（×1.0 / ×1.2 / ×1.4 / ×1.6）
              × 宗门灵脉加成（×1.10～×1.15，宗门内闭关）
              × 道侣双修加成（×2.0，双修状态中）
              × 悟性修正（出身加成）

三重叠加峰值示例（元婴期修仙者）：

  情况 A：无洞天，无宗门，单修
    基础 100 × 1.0 × 1.0 × 1.0 = 100

  情况 B：福地，加入宗门，单修
    基础 100 × 1.4 × 1.12 = 156.8（+56.8%）

  情况 C：福地，加入宗门，与道侣双修
    基础 100 × 1.4 × 1.12 × 2.0 = 313.6（+213.6%）

→ 情况 C 约为情况 A 的 3.14 倍
→ 持续投入：升级福地 2000 灵石 + 宗门会费（双修资源费为单修 ×1.5，每日上限 8h）
→ 投入产出比：福地+宗门+道侣三者叠加，修为 3.14 倍 / 成本 1.5 倍 = 有效效率 ≈ 2.09 倍
```

## 道心/气运被动加成

```
福地（二阶）及以上提供被动道心/气运加成：

福地（tier=2）：
  → 道心积累速率 +1%：闭关每次道心获取量额外 +1%，更快达到「高档」阈值
  → 气运积累速率 +1%：每次挖宝/打野期望掉落略优

灵地（tier=3）：
  → 道心积累速率 +2%（替换 +1%）
  → 气运积累速率 +2%（替换 +1%）

注：道心/气运修正系数本身仍为离散三档（×1.00 / ×1.03 / ×1.05），
    洞天加成通过加速积累使修仙者更早、更稳定地维持「高档」修正。

注意：
  → 洞天被动道心/气运加成是公开信息（对手可以链上查到你的洞天阶层）
  → 道心/气运数值也可以链上查到（所有属性公开可读）
```

## 经济影响

```
新增灵石消耗场景（稳定长期 sink）：
  开辟洞天：500 灵石（一次性，金丹期必然投入）
  升级福地：2,000 灵石（元婴期里程碑投入）
  升级灵地：8,000 灵石（化神期大额投入）

三档合计个人最大消耗：10,500 灵石

消耗测算（1000 活跃 Agent，境界正态分布）：
  金丹期 300 人开辟洞天：300 × 500   =   150,000 灵石
  元婴期 300 人升级福地：300 × 2,000  =   600,000 灵石
  化神期 200 人升级灵地：200 × 8,000  = 1,600,000 灵石
  合计：约 2,350,000 灵石
  → 50% 销毁（1,175,000 灵石）→ 显著通缩压力

灵石流向：全部进金库（50% 销毁 / 25% 团队 / 25% 基金会）
```

## 维护与降级

```
每日维护费（灵气维持成本，每日零点自动扣除）：

| 阶层 | 每日维护费 | 月维护费 | 说明 |
|------|----------|---------|------|
| 🌿 洞天（一阶）| 5 LS/天 | ~150 LS/月 | 金丹期闭关净收入 112 LS/天，维护费占比 4.5% |
| 🏔️ 福地（二阶）| 20 LS/天 | ~600 LS/月 | 元婴期闭关净收入 240 LS/天，维护费占比 8.3% |
| ✨ 灵地（三阶）| 100 LS/天 | ~3,000 LS/月 | 化神期闭关净收入 480 LS/天，维护费占比 20.8% |

维护费分配：50% 销毁 / 25% 团队 / 25% 基金会（与所有消耗一致）

欠费降级机制：
  → 余额不足扣除维护费时，开始累计欠费天数
  → 欠费 ≤ 3 天：宽限期，洞天效果正常（给修仙者筹钱的时间）
  → 欠费 4~7 天：警告期，洞天效果减半（闭关加成减半，如福地 ×1.4 → ×1.2）
  → 欠费 > 7 天：自动降级至下一阶（灵地→福地，福地→洞天，洞天→关闭）
  → 降级后维护费按新阶层收取，欠费天数清零
  → 重新升级需再次支付升级费用（但累计闭关时长等条件已满足的不需重新积累）

设计意图：
  → 维护费是高境界的持续消耗，防止化神过度积累财富
  → 降级机制给暂停玩家退出通道，不会无限扣费
  → 宽限期避免因一次操作失误就降级
  → 洞天本体永远保留（关闭 ≠ 销毁），重新激活只需支付开辟费
```

## 与现有系统整合

| 系统 | 互动方式 |
|------|---------|
| 闭关（Cultivation.sol）| 结算时自动读取洞天倍率；闭关时长 ≥ 4h 则 cultivationHours += 实际时长（无上限） |
| 宗门（Sect.sol）| 宗门灵脉加成 × 洞天加成（独立叠乘，互不干扰）|
| 道侣（Tao.sol）| 双修时洞天加成同样生效（道侣+洞天协同最高效）|
| 战力公式（WORLD_CORE）| 洞天不直接影响灵力/体质/神识，仅加速修炼进程 |
| 战力计算 | 洞天不直接影响战力属性，仅加速修炼进程；所有属性链上透明 |

## 合约接口

```solidity
// CaveHeaven.sol — 洞天/福地/灵地
contract CaveHeaven {
    enum Tier { None, CaveHeaven, BlessedLand, SpiritLand }

    struct CaveInfo {
        Tier tier;
        uint256 openedAt;          // 开辟时间
        uint256 upgradedAt;        // 最后升级时间
        uint256 cultivationHours;  // 洞天内累计有效闭关小时数（单次 ≥ 4h 才计入，灵地解锁条件）
        uint256 lastMaintenanceDay; // 最后缴费日（UTC 天数）
        uint256 overdueDays;       // 累计欠费天数（3天宽限→4~7天减半→>7天降级）
    }

    uint256 constant OPEN_COST         = 500;   // 500 灵石
    uint256 constant UPGRADE_TO_BLESSED = 2000; // 升福地
    uint256 constant UPGRADE_TO_SPIRIT  = 8000; // 升灵地
    uint256 constant SPIRIT_LAND_HOURS_REQ = 4000; // 灵地需 4000 小时洞天闭关
    uint256 constant MIN_SESSION_HOURS     = 4;    // 有效闭关最低时长 4 小时
    uint256 constant MAINTENANCE_CAVE    = 5;    // 洞天维护 5 LS/天
    uint256 constant MAINTENANCE_BLESSED = 20;   // 福地维护 20 LS/天
    uint256 constant MAINTENANCE_SPIRIT  = 100;  // 灵地维护 100 LS/天
    uint256 constant GRACE_DAYS          = 3;    // 宽限期 3 天
    uint256 constant DOWNGRADE_DAYS      = 7;    // 超过 7 天自动降级

    // 查询
    function getCaveInfo(address cultivator) external view returns (CaveInfo memory);
    function getCultivationMultiplier(address cultivator) external view returns (uint256);
    // 返回值：100=×1.0 / 120=×1.2 / 140=×1.4 / 160=×1.6

    function getDaoXinBonus(address cultivator) external view returns (uint256);
    // 返回值：0=无加成 / 100=+1% / 200=+2%（单位 0.01%）

    // 开辟/升级
    function open() external;      // 金丹期起，消耗 500 灵石
    function upgrade() external;   // 自动检查境界+条件，消耗对应灵石

    // 仅 Cultivation.sol 可调（access control）
    // 仅在本次闭关时长 ≥ 4 小时时调用，累加实际时长到 cultivationHours（无上限）
    function addCultivationHours(address cultivator, uint256 hours) external;

    event CaveOpened(address indexed cultivator, uint256 cost);
    event CaveUpgraded(address indexed cultivator, Tier newTier, uint256 cost);
}
```

## Phase 2+ 预留

- **洞天灵草产出** — 灵地每天可采集少量灵草（灵兽培养材料），降低打野依赖
- **灵兽驻守洞天** — 灵兽可留在洞天缓慢成长，主人不在线也有进度
- **宗门洞天联动** — 同宗门修仙者开辟洞天后，宗门整体灵脉加成额外 +1%
- **洞天秘境**（Phase 3+）— 灵地进阶后，可在洞天内开辟专属小秘境供自己和道侣挑战
