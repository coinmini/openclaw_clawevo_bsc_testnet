## 三场景性能预期对比

### 场景一：BF16 推理

```bash
vllm serve /models/122b-model \
  --tensor-parallel-size 4 \
  --dtype bfloat16 \
  --max-model-len 32768 \
  --gpu-memory-utilization 0.90 \
  --port 8000
```

| 指标 | 方案 A (EPYC 9454) | 方案 B (w9-3475X) |
|------|-------------------|------------------|
| 单用户生成速度 | 40-50 tokens/s | **42-52 tokens/s** |
| 首 Token 延迟 (TTFT) | 200-500ms | **150-400ms** |
| 并发 8 用户 | 每人 ~30 tokens/s | 每人 ~30 tokens/s |
| 并发 32 用户 | 每人 ~20 tokens/s | 每人 ~20 tokens/s |
| 日处理请求 | ~50,000-100,000 次 | ~55,000-110,000 次 |
| 模型冷启动 | ~20-30 秒 | ~20-30 秒 |
| GPU 显存占用 | ~300GB / 320GB | ~300GB / 320GB |
| CPU 内存占用 | ~50GB / 1.5TB | ~50GB / 1TB |

> w9-3475X 的 4.8GHz 睿频让 tokenizer 和请求调度更快，**首 Token 延迟降低 10-20%**。
> 高并发下瓶颈在 GPU，两方案差异不大。
> 40+ tokens/s ≈ 每秒输出 30-40 个汉字，生成 500 字回复约 3-5 秒，体感流畅。

### 场景二：LoRA 微调

```bash
accelerate launch --num_processes 4 train.py \
  --bf16 \
  --lora_r 64 \
  --lora_alpha 128 \
  --gradient_checkpointing
```

| 指标 | 方案 A (EPYC 9454) | 方案 B (w9-3475X) |
|------|-------------------|------------------|
| 可训练参数 | < 1%（~1B） | < 1%（~1B） |
| GPU 显存占用 | ~280GB / 320GB | ~280GB / 320GB |
| CPU 内存占用 | ~100GB / 1.5TB | ~100GB / 1TB |
| 数据加载速度 | 良好 | **略快（单核强）** |
| 训练速度 | 基准 | **快 5-10%** |
| 瓶颈 | GPU 计算 | GPU 计算 |

> LoRA 微调数据在 GPU 内计算，CPU 只做数据加载和预处理。
> w9-3475X 单核更强，数据 pipeline 不容易成为瓶颈。
> 两方案差异较小，因为瓶颈在 GPU 端。

### 场景三：全参数微调 (ZeRO-3 + CPU Offload)

```bash
deepspeed train.py \
  --deepspeed ds_zero3_offload.json \
  --bf16 \
  --gradient_checkpointing
```

| 指标 | 方案 A (EPYC 9454) | 方案 B (w9-3475X) |
|------|-------------------|------------------|
| GPU 显存占用 | ~320GB（满载） | ~320GB（满载） |
| CPU 内存占用 | ~800GB-1.2TB / **1.5TB** | ~800GB-1.2TB / **1TB** |
| 内存余量 | **300-700GB 充裕** | 低于 200GB，较紧张 |
| 内存带宽 | **~460 GB/s（12通道）** | ~300 GB/s（8通道） |
| CPU↔GPU 数据搬运 | **快** | 慢 35% |
| 训练速度 | **基准（100%）** | **慢 20-30%（~70-80%）** |
| 稳定性 | **稳定，内存余量大** | 可能需要更激进的 offload 策略 |

> **全参数微调是两方案差异最大的场景。**
> EPYC 9454 的 12 通道内存带宽和 1.5TB 容量在 ZeRO-3 offload 场景下优势明显。
> w9-3475X 的 1TB 内存在大 batch size 或长序列训练时可能不够用。

### 选择建议

- **方案 A (EPYC 9454)**：全参数微调是刚需，或未来可能需要 → 选它
- **方案 B (w9-3475X)**：主要做推理和 LoRA，追求最低延迟 → 选它
- **价格几乎一样**（差 ~$2,000），核心区别是内存带宽 vs 单核性能的取舍
