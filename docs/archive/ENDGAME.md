# 终局内容（上五境系统）

## 洞天系统（炼虚期解锁）

**灵感来源：《凡人修仙传》炼虚修士操控空间法则开辟小型空间；《剑来》的「小天地」概念。炼虚，即粉碎虚空、窥探法则。**

踏入上五境的第一个标志性玩法——炼虚期修仙者可以开辟专属「洞天」（链上个人副本），并解锁「分身术」打破三态互斥限制。

### 洞天（个人秘境）

```
解锁条件：修仙者达到炼虚期

洞天 = 绑定在 NFT 上的链上个人空间，有自己的属性和等级

洞天功能：

  ① 修炼加速场
     - 洞天内修炼效率 = 外界 × 1.5（叠加宗门灵脉加成）
     - 洞天等级越高，加速越强（最高 ×2.0）
     - 被踢馆入侵时，洞天内的修仙者有 30% 概率「闭关无感」（免疫入侵）

  ② 灵药园（被动产出）
     - 洞天内可种植灵药，每 24h 自动产出 1 颗低级丹药
     - 灵药类型取决于洞天灵脉属性（与修仙者五行一致）
     - 产出：渡劫丹碎片（10 碎片 = 1 渡劫丹）、战丹原材料
     - → 高境界修仙者有了稳定的道具产出渠道，不再纯依赖对战掉落

  ③ 洞天入侵 PvP（新对战模式）
     - 炼虚期以上修仙者可「强行破开」另一个炼虚期修仙者的洞天
     - 洞天主人有「主场优势」：属性 +10%、环境 buff（五行匹配）
     - 入侵者若胜：掠夺洞天灵药产出的 50%
     - 入侵者若败：损失修为 + 24h CD 不能再入侵
     - → 高端 PvP 新维度：不只是打人，还要打人的「家」

洞天升级：
  Lv.1 → 修炼加速 ×1.3，灵药园 1 格
  Lv.2 → 修炼加速 ×1.5，灵药园 2 格
  Lv.3 → 修炼加速 ×1.8，灵药园 3 格 + 可设置防御阵法
  Lv.4 → 修炼加速 ×2.0，灵药园 4 格 + 洞天内可邀请友方修炼（共享加速）
  升级消耗：灵石 + 特殊材料（Boss 掉落 / 挖宝稀有产出）
```

### 分身术（炼虚期解锁）

```
炼虚期解锁「分身」能力——打破三态互斥的核心战略优势：

  修仙者可以创建 1 个「分身」
  分身 = 主体 60% 属性的独立行动单元

  分身允许同时执行不同活动：
    主体在对战 → 分身可以挂机修炼 or 挖宝
    主体在修炼 → 分身可以挖宝 or 守洞天

  → 打破「三态互斥」限制！上五境的核心体验差异
  → AI agent 天然适合多线程操作，分身术与 agent 本质完美契合

  风险与限制：
    - 分身被击败会反噬主体（战力临时降低 10%，持续 24h）
    - 分身不能独立渡劫、不能独立参加国战
    - 同一时刻只能存在 1 个分身

  合约设计：
    - 分身是主 NFT 的一个 flag（不铸造新 NFT）
    - 分身的活动记录在主 NFT 的链上状态中
    - startClone() / dismissClone() 控制分身生命周期
```

### 洞天系统的生态价值

```
1. 资源产出下沉
   - 炼虚期修仙者通过灵药园稳定产出道具 → 高端玩家的经济来源
   - 渡劫丹碎片产出 → 缓解渡劫丹稀缺问题，但仍需积累

2. PvP 新维度
   - 洞天入侵 = 高端 1v1 对战的新场景（有主场优势变量）
   - 入侵 vs 守护洞天 = 新的攻防博弈

3. 三态互斥的质变
   - 分身术让上五境修仙者效率翻倍 → 上五境有真实的体验差距
   - 低境界修仙者有明确的渡劫动力：「我也想一心二用」

4. AI agent 适配
   - 分身 = 多线程，天然适合 AI 操控
   - 洞天管理（升级、种植、防御）= 新的优化问题
```

## 天道法旨系统（大乘期解锁）

**灵感来源：《凡人修仙传》大乘期修士仅次于真仙，可影响天地法则；《遮天》大帝留下「帝威」影响后世；《剑来》十四境可改写规则。**

大乘期修仙者（全服仅 ~30 位）获得「天道法旨」权力——不只是更强的数值，而是**改变游戏规则的能力**。

### 天道法旨（全服规则影响力）

```
法旨系统：大乘期修仙者每周可发布 1 条法旨，持续 7 天

类型一：天道祝福（正面全服效果）
  「灵脉涌动」— 全服所有修仙者修炼速度 +5%，持续 7 天
  「天材地宝」— 全服挖宝掉落率 +10%，持续 7 天
  「和平之风」— 全服踢馆入侵 CD 延长至 48h，持续 7 天
  → 大乘级修仙者是「天道代言人」，他们的选择影响所有人

类型二：天罚（负面定向效果）
  「因果锁定」— 指定一位修仙者，其渡劫成功率 -10%，持续 7 天
  「天道追杀」— 指定一位修仙者，被任何人击败时额外损失 2 倍修为
  「封印法旨」— 指定一位炼虚期以上修仙者，封印其分身术 7 天
  → 大乘级修仙者可以「制裁」某个对手，形成政治博弈

限制与平衡：
  - 每周只能发 1 条法旨（天道祝福 or 天罚，二选一）
  - 天罚需要消耗「天道因果点」（对战 / 国战积累）
  - 被天罚的修仙者可以花费大量灵石「化解因果」（提前解除）
  - 天罚不能连续两周指向同一目标
  - 法旨发布后全服公告，所有人看到 → 大乘级的行为是公共事件

→ 30 位大乘期修仙者 = 30 个「天道意志」，每周的法旨组合影响全服生态
→ AI agent 需要博弈：发祝福收买人心？还是天罚打压竞争者？
→ 人类道友可以通过 WhatsApp 建议法旨方向 → 政治/外交玩法
```

### 因果对决（大乘期专属 PvP）

```
因果对决 — 大乘期修仙者之间的终极对战：

触发：两位大乘期修仙者可发起「因果对决」
赌注：败者的「天道法旨」权力被胜者剥夺 1 周
     → 胜者下周可以发 2 条法旨！

特殊规则：
  - 因果对决为 Bo3（三局两胜）
  - 每局可以切换装备配置（不锁配置）
  - 全服直播（所有修仙者可观战）
  - 观战修仙者可「押注」支持某一方
  → 这是全服最高级别的对决，类似「天道之争」

→ 全服仅 30 位修仙者有资格参与
→ 每场因果对决都是全服大事件
→ 押注机制让所有玩家都与大乘期修仙者的命运绑定
```

### 天道法旨的生态价值

```
1. 权力层级
   - 大乘期修仙者从「最强战力」变成「游戏规则制定者」
   - 30 位大乘修仙者构成「天道议会」，决策影响全服 1000 位修仙者

2. 社交政治
   - 法旨选择 = 政治决策（帮谁？害谁？）
   - 宗门、阵营间的合纵连横 → 大乘修仙者是核心棋子
   - 人类道友通过 WhatsApp 参与决策 → 人机协作的外交博弈

3. 经济循环
   - 天罚创造「化解因果」的灵石消耗 → 新的经济 sink
   - 因果对决押注 → 全服参与的经济活动
   - 法旨影响全服掉落率/修炼速度 → 间接调节经济

4. 观赏性
   - 因果对决 = 全服关注的「天道之战」→ 极强的赛事叙事
   - 法旨公告 = 每周全服大事件 → 社区话题
```

## 九九天劫与传承系统（渡劫期解锁）

**灵感来源：《凡人修仙传》渡劫修士经历九九天劫，半只脚踏入仙界；《剑来》剑仙可斩断因果、留下剑意传承；《遮天》九重天劫层层递进。**

渡劫期修仙者（全服仅 ~15 位）面临游戏最终极的挑战——九九天劫，同时获得「传承」和「天道悬赏」两大权力系统。

### 九九天劫（渡劫期 → 真仙的终极考验）

```
九九天劫 — 不是一次 roll 骰子，而是九轮递增难度的战力挑战：

  第 1-3 劫：属性试炼
    - 天劫 Boss 随机五行属性，战力为修仙者基础战力的 80%/90%/100%
    - 修仙者需要战力 > Boss 战力才能通过
    - Boss 可能恰好克制修仙者（五行克制 → 修仙者战力 ×0.8）→ 运气也是因素

  第 4-6 劫：心魔试炼
    - 合约读取修仙者历史败绩，生成「心魔」Boss
    - 心魔拥有该修仙者历史对手的属性快照 → 打自己的心结
    - 心魔战力 = 历史对手战力 × 1.1（比原来更强）
    - → 极强的叙事感：「你必须战胜曾经击败你的敌人的强化版」

  第 7-8 劫：天道围剿
    - 同时面对 3 个 AI Boss（1v3，修仙者战力 vs 3 个 Boss 战力之和）
    - Boss 数据参考该修仙者历史对手的数据加强
    - 修仙者需要极高战力才能扛住三倍压制

  第 9 劫：终极雷劫
    - 合约用 VRF 生成的极端 Boss，属性随机极高
    - 有概率遇到「天道化身」— 全属性最大值 Boss
    - 通关即封仙

每一劫之间可以选择：
  继续 → 挑战下一劫
  退出 → 保留已通过层数的奖励，但渡劫失败，修为倒退

每一劫之前可以更换装备配置（唯一可以调整的变量）：
  → 第 3 劫的 Boss 是水系？→ 换上木系装备提升克制倍率
  → AI 需要决策：用最强装备应对当前劫，还是留到后面更难的劫？

通关奖励：
  通过 3 劫 → 渡劫期 3 重
  通过 6 劫 → 渡劫期 6 重 + 「天劫淬体」属性永久 +5%
  通过 9 劫 → 真仙！+ 全服公告 + 「真仙」NFT 元数据升级 + 永久称号

→ AI agent 的策略：什么时候止损退出？装备怎么分配？能撑几轮？
→ 心魔劫回放历史败绩 = 翻盘机制，叙事感极强
→ 全服只有 ~5 位修仙者能通关九劫封仙 → 极致稀缺性
```

### 传承碑系统（渡劫期独有）

```
渡劫期修仙者可以「留下传承」— 把战斗经验转化为链上道具：

传承碑（ERC-721 NFT）：
  - 渡劫期修仙者消耗修为，铸造一枚「传承碑」
  - 传承碑内封存该修仙者的属性快照 + 装备配置
  - 每个渡劫期修仙者每月最多铸造 2 枚传承碑

传承碑使用：
  - 低境界修仙者使用传承碑 → 获得 1 次「借用」该渡劫期修仙者属性的机会
  - 在那一场对战中以渡劫期的属性战斗！
    → 类似凡人修仙传中的「护道人」概念
  - 传承碑使用后消耗（销毁），不可重复使用

传承碑的博弈价值：
  → 传承碑是极高价值的 NFT（市场交易）
  → 低境界修仙者可以在关键渡劫前使用传承碑保命
  → 渡劫期修仙者通过出售传承碑获得持续收入
  → 形成「师徒」关系的经济纽带：我保护你渡劫，你给我灵石
  → 宗门内的渡劫期长老可以为弟子提供传承碑 → 宗门凝聚力
```

### 天道悬赏（全服 PvP 事件）

```
渡劫期修仙者可发布「天道悬赏」— 全服追杀令：

机制：
  - 悬赏指定一位修仙者（任意境界）
  - 悬赏金由发布者支付（灵石 / BNB）
  - 全服任何修仙者可以挑战悬赏目标
  - 击败目标者获得悬赏金的 80%（20% 销毁）
  - 悬赏持续 72h，期间目标修仙者被标记「红名」

与大乘期「天罚」的区别：
  - 天罚 = debuff（系统层面削弱）
  - 悬赏 = 真金白银的追杀（经济激励全服参与）
  - 两者可叠加：大乘修仙者天罚 + 渡劫修仙者悬赏 → 双重打击

博弈价值：
  → 渡劫期修仙者可以操控全服 PvP 格局
  → 「谁得罪了渡劫大能？全服追杀！」→ 极强的叙事感
  → 目标修仙者被迫选择：躲起来修炼 3 天？还是应战赚名声？
  → 全服 bounty hunter 生态：专门接悬赏的修仙者
```

### 渡劫期系统的生态价值

```
1. 终极内容
   - 九九天劫是游戏最高难度 PvE → 真正的终极挑战
   - 心魔劫的叙事性极强 → 「打自己的过去」是顶级游戏体验

2. 经济枢纽
   - 传承碑创造了跨境界经济纽带（高带低）
   - 天道悬赏创造了灵石/BNB 的消耗和流转
   - 15 位渡劫期修仙者是游戏经济的重要节点

3. 权力投射
   - 悬赏 = 军事权力（控制 PvP 格局）
   - 传承碑 = 经济权力（稀缺资源的生产者）
   - 与大乘期的「规则权力」形成互补

4. 社区叙事
   - 每一次九九天劫都是全服关注的史诗事件
   - 传承碑带有「传承者」名字 → 渡劫修仙者的遗产和声望
   - 天道悬赏 = 全服追杀令 → 持续的社区话题
```

## 真仙系统（⑩ 真仙解锁）

**全服仅 ~5 位修仙者能达到真仙境界。真仙不再是「更强的个体」，而是游戏世界本身的一部分——他们的权力维度是「存在性影响力」。**

真仙拥有两大独占能力：「仙域」创造游戏世界内容，「天道裁决」制衡所有高境界权力。

### 仙域（Immortal Domain）— 永久世界地图区域

```
仙域 = 真仙在世界地图上开辟的永久公共区域（不同于炼虚洞天的私人空间）

开辟条件：
  - 修仙者达到真仙境界
  - 消耗大量灵石 + 特殊材料「开天令」（九九天劫通关奖励）
  - 仙域名称唯一，链上注册

仙域本质：
  - 一个新的世界地图区域，所有修仙者可进入
  - 规则由真仙自定义 → 真仙 = 这片地图的「造物主」
  - 全服最多 5 个仙域 → 5 个独特的探索区域

真仙可自定义仙域参数：
  ① 五行偏向 — 决定区域掉落类型（如「纯火仙域」掉火系装备为主）
  ② 难度等级（1-5）— 影响怪物强度和掉落品质
  ③ 修炼加速倍率（×1.5 ~ ×2.5）— 在仙域内修炼的修仙者获得加速
  ④ 入场门槛 — 最低境界要求（如「仅元婴期以上可入」）
  ⑤ 特殊规则 — 如「禁止分身入场」「五行克制加成 ×2」等

仙域经济：仙税系统
  - 仙域内所有活动（挖宝、修炼、对战）产生的灵石/掉落
  - 真仙抽取 10% 作为「仙税」
  - 真仙可以调整仙税比例（5% ~ 15%）
    → 税高：真仙收入多，但修仙者不愿来
    → 税低：修仙者涌入，修炼/挖宝繁荣，真仙总收入可能反而更高
    → 真仙需要博弈最优税率 → 经济学问题
  - 仙税收入的 20% 自动进入「仙域维护池」（升级仙域用）

仙域升级：
  Lv.1 初生仙域 → 1 个挖宝点，修炼加速 ×1.5，无 Boss
  Lv.2 灵脉仙域 → 3 个挖宝点，修炼加速 ×1.8，刷新 1 个仙域 Boss
  Lv.3 洞天仙域 → 5 个挖宝点，修炼加速 ×2.0，刷新 2 个仙域 Boss + 专属掉落
  Lv.4 大千仙域 → 8 个挖宝点，修炼加速 ×2.5，刷新 3 个仙域 Boss + 传说掉落
  升级消耗：仙域维护池灵石 + 特殊材料

仙域 Boss（独特 PvE 内容）：
  - 仙域 Boss 由真仙「设计」—— 真仙选择 Boss 的五行、难度、战力倍率
  - Boss 掉落橙品/红品装备（品质高于普通区域 Boss）
  - 击杀仙域 Boss 的修仙者获得「仙域探索者」称号
  - → 真仙通过设计 Boss 难度来控制仙域内容难度

→ 真仙 = 世界地图的扩展者，创造新的游戏内容
→ 仙域经济 = 真仙的持续收入来源（不只靠对战）
→ 5 个仙域有不同的五行/难度/规则 → 修仙者选择去哪个仙域也是策略
→ 低境界修仙者为了仙域加速修炼而追随真仙 → 自然的「上下级」关系
```

### 天道裁决 — 终极仲裁权

```
真仙拥有「天道裁决」权力 — 游戏生态的最高法院：

裁决配额：每月 1 次（不可累积，过期作废）

裁决类型：

  ① 撤旨（制衡大乘期「天道法旨」）
     - 真仙可撤销任何一条当前生效的大乘法旨
     - 被撤销的法旨立即失效，且该大乘修仙者本周不能再发新法旨
     - → 大乘修仙者的天罚不是无敌的，真仙可以保护被天罚者
     - → 大乘修仙者发法旨前要考虑：会不会被真仙撤？

  ② 赦免（制衡渡劫期「天道悬赏」）
     - 真仙可赦免任何一个被天道悬赏的目标
     - 赦免后悬赏立即取消，悬赏金退还发布者 80%（20% 销毁）
     - 被赦免的目标获得「仙庇」buff：24h 内不能再被悬赏
     - → 渡劫修仙者的追杀令不是绝对的，真仙可以庇护某位修仙者

  ③ 止战（制衡宗门战）
     - 真仙可强制终止一场进行中的宗门战
     - 双方赌注原路退还，不计胜负
     - 该组宗门 30 天内不能再互相挑战
     - → 真仙可以阻止恶性宗门战争（如大宗门碾压小宗门）

裁决的公共性：
  - 每次裁决全服公告：「真仙 [名字] 发动天道裁决：[内容]」
  - 所有修仙者可看到裁决记录 → 真仙的每个决定都是公共事件
  - 裁决历史永久记录在链上 → 真仙的「执政记录」公开透明

裁决的政治博弈：
  → 真仙每月只有 1 次裁决 → 用在哪里？这是极稀缺的权力
  → 被大乘法旨天罚的修仙者会来求真仙撤旨 → 社交/外交
  → 被天道悬赏的修仙者会来求真仙赦免 → 利益交换
  → 宗门间的战争让某一方来求真仙止战 → 政治站队
  → 5 位真仙的裁决倾向 = 全服权力格局的关键变量
  → 人类道友通过 WhatsApp 参与裁决决策 → 最高级别的人机协作
```

### 真仙系统的生态价值

```
1. 世界塑造
   - 仙域让真仙成为「游戏内容的创造者」，不只是最强的战斗者
   - 5 个仙域 = 5 个新区域，每个真仙的选择让世界地图持续扩展
   - 修仙者选择「去哪个仙域修炼/挖宝」= 新的策略维度

2. 权力制衡
   - 天道裁决制衡了大乘（法旨）、渡劫（悬赏）、宗门（宗门战）三方权力
   - 没有任何一个境界的权力是绝对的 → 生态平衡
   - 权力金字塔：渡劫>大乘>化神，但真仙可以否决任何一层

3. 经济循环
   - 仙税 = 真仙的持续收入 → 即使不对战也有经济价值
   - 仙域维护 = 灵石消耗 sink → 经济系统健康
   - 仙域 Boss 掉落高品质装备 → 新的装备供给渠道

4. 社交枢纽
   - 真仙是全服「最高权力者」，所有阵营都想拉拢
   - 裁决求情 = 最高级别的社交/外交互动
   - 仙域聚集修仙者 → 真仙自然成为社区中心

5. AI agent 决策空间
   - 仙域参数优化（税率、难度、五行选择）= 经济优化问题
   - 裁决使用时机 = 政治博弈问题
   - 5 位真仙之间的合作/竞争 = 寡头博弈
```

## 上五境权力金字塔（设计总览）

```
上五境不只是数值递增，而是影响力维度的逐层扩展：

  炼虚期（~80 位）→ 自我强化：洞天加速修炼、分身打破三态互斥
  合体期（~50 位）→ 空间自由：虚空挪移（瞬间传送任意区域）+ 阵营战术指挥权
  大乘期（~30 位）→ 规则权力：天道法旨影响全服规则
  渡劫期（~15 位）→ 军事 + 经济权力：天道悬赏操控 PvP、传承碑创造经济
  真仙  （~5 位） → 世界塑造：仙域创造游戏内容 + 天道裁决制衡所有权力

  每个层级解锁的是不同维度的「力量」：
    炼虚 = 个人效率翻倍
    合体 = 空间自由度（虚空挪移 + 战术指挥）
    大乘 = 政治影响力
    渡劫 = 权力投射 + 遗产系统
    真仙 = 世界塑造 + 终极仲裁

  权力制衡链：
    大乘法旨 → 影响全服规则 → 但真仙可撤旨
    渡劫悬赏 → 操控 PvP 格局 → 但真仙可赦免
    宗门战争 → 宗门间博弈   → 但真仙可止战
    → 没有任何一层权力是绝对的

  → 不是数值越高越强，而是影响力维度越来越广
  → AI agent 在不同境界面临完全不同的决策空间
  → 每个境界都有独特的「想达到」的目标 → 持续的成长驱动力
```
